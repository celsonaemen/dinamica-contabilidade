# Dinâmica Contabilidade - Site Estático Inicial

Base inicial do site institucional da Dinâmica Contabilidade.

## Como abrir

Abra o arquivo `index.html` no navegador.

Caminho:

`C:\Users\Dinamica\Saved Games\dinamica-contabilidade\index.html`

## Arquivos principais

- `index.html` - estrutura da página.
- `styles.css` - visual, responsividade e identidade.
- `script.js` - menu mobile e configuração futura do WhatsApp.
- `public/logo.png` - logo usada no site.
- `docs/` - documentos de produto, arquitetura e design system.
- `.ai/skills/dinamica-contabilidade/SKILL.md` - contexto do projeto para IA/Codex.

## Configurar depois

No arquivo `script.js`, preencher:

```js
const WHATSAPP_NUMBER = "55DDDNUMERO";
```

No arquivo `index.html`, trocar o e-mail provisório:

`contato@dinamicacontabilidade.com.br`

Também falta confirmar:

- Telefone oficial.
- Endereço ou cidade de atendimento.
- Redes sociais.
- Depoimentos reais.
- CNPJ, se for exibido no rodapé.
